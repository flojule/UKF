## Overview

This repository implements a full **Unscented Kalman Filter (UKF)** pipeline for 2D mobile robot localization on the [UTIAS MRCLAM dataset](http://asrl.utias.utoronto.ca/datasets/mrclam/). The robot state is $(x, y, \theta)$ and the filter fuses wheel odometry with range-and-bearing measurements to known landmarks.

### Motion model

The robot moves on a plane following unicycle kinematics. If $\omega \neq 0$ it traces a circular arc; if $\omega = 0$ it moves in a straight line:

$$
\begin{pmatrix} x \\ y \\ \theta \end{pmatrix}_{t} =
\begin{cases}
\begin{pmatrix}
x_{t-1} - \tfrac{v}{\omega}\bigl(\sin\theta_{t-1} - \sin(\theta_{t-1}+\omega\Delta t)\bigr) \\
y_{t-1} + \tfrac{v}{\omega}\bigl(\cos\theta_{t-1} - \cos(\theta_{t-1}+\omega\Delta t)\bigr) \\
\theta_{t-1} + \omega\Delta t
\end{pmatrix} & \omega \neq 0 \\[1em]
\begin{pmatrix}
x_{t-1} + v\Delta t\cos\theta_{t-1} \\
y_{t-1} + v\Delta t\sin\theta_{t-1} \\
\theta_{t-1}
\end{pmatrix} & \omega = 0
\end{cases}
$$

### Measurement model

The sensor returns range and bearing to visible landmarks. For a landmark at $(\ell_x, \ell_y)$:

$$
\mathbf{z} = \begin{pmatrix}
\sqrt{(\ell_x - x)^2 + (\ell_y - y)^2} \\
\operatorname{atan2}(\ell_y - y,\; \ell_x - x) - \theta
\end{pmatrix}
$$

### UKF results on MRCLAM ds0

| Metric | UKF | Dead reckoning |
|---|---|---|
| Avg. position error | **0.107 m** | diverges |
| Avg. bearing error  | **0.049 rad** | diverges |

Tuned parameters: $P_0 = \mathrm{diag}(10^{-6}, 10^{-6}, 10^{-6})$, $Q_0 = \mathrm{diag}(10^{-6}, 10^{-6}, 3.6\times10^{-5})$, $R_0 = \mathrm{diag}(10^{-2}, 10^{-2})$, $\alpha = 0.1$, $\kappa = 0$, $\beta = 2$.

numpy

Install with `pip install numpy matplotlib`.

## Running

```bash
python src/run.py
```

Both Part A and Part B run by default (`partA = True`, `partB = True` in `main()`).

Key flags in `main()`:

| Flag | Default | Effect |
|---|---|---|
| `i` | `0` | Dataset index (`0` → `data/ds0/`, `1` → `data/ds1/`) |
| `use_resampled` | `False` | Use pre-resampled 50 Hz data from `data/ds0_RS/` |
| `export` | `False` | Resample ds0 and write results to `data/ds0_RS/` |
| `DEBUG` | `False` | Truncate data to 2000 steps for fast iteration |
